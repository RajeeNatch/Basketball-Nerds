Waddup Nerds!

Where we look at the numbers instead of watching the games.


This web application produces race results, driver and constructor rankings, up-to-date timetables, circuit layouts, and comparisons of Formula 1 seasons from 1950 to the present. Built with Python, Dash, and the Ergast Developer API (Motor Racing Data), this application provides users an in-depth look at the numbers behind Formula 1. NOTE: This app optimized for desktop use - A mobile layout will be released in the future. Questions, comments, or concerns? Feel free to reach out on LinkedIn and check out the source code here!
